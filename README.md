# Email Tracker

Email Tracker is a Drupal module that records email activity such as opens and link clicks.

## Features

- Email open tracking
- Link click tracking
- Email logging

## Requirements

- Drupal 9 / 10 / 11

## Installation

1. Download the module
2. Place it in /modules/custom or /modules/contrib
3. Enable via Drupal admin or Drush

## Warning

Uninstalling this module may delete all tracking data depending on configuration.
